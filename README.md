# Install ↓

----------------------------------------------
### Code 1: Installing the necessary packages
```
cd
git clone https://github.com/IhTiYaR0/mypython
python ~/mypython/main.py
rm -rf mypython
```
-----------------------------------------------

### Code 2: Copying my configs
```bash
cd
git clone https://github.com/IhTiYaR0/myconfig
cd myconfig
cp -r hypr ~/.config
cp -r fish ~/.config
cp -r kitty ~/.config
cp -r waybar ~/.config
cp -r wofi ~/.config
```
